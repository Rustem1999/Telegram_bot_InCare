1. Cохраняете все файлы в одном папке (также в эту же папку потребуется файл "serviceAccountKey.json").
2. Необходимо установить Python библиотеки по списке, находящийся в файле "python_libraries.txt". Можно установить библиотеки на терминале Windows PowerShell с помощью команды "pip install -r python_libraries.txt" находясь в папке проекта.
3. В файле "main.py" вставляете токен в значение переменной "bot_token" и запускаете программу. Переменная выглядит следующим  образом:
"
	# Замените на свой токен
    	bot_token = 'Вставляете токен сюда!!!' 
"
